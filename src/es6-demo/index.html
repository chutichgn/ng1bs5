ng1bs5 ES6 Demo

Bootstrap 5 Components for AngularJS with ES6 Modules

This demo imports ng1bs5 components directly from source using ES6 modules

üì¶ ES6 Module Demo

This demo application uses ES6 imports to load ng1bs5 components directly from the source (src/components/). It's built with Webpack and demonstrates how to use the library in modern JavaScript applications.

Important: This demo uses ng1bs5 directives (like bs-popover, bs-tooltip) instead of native Bootstrap JavaScript. All Bootstrap functionality is provided by ng1bs5.

Modal Component

Dialog boxes and popup windows with ng1bs5.

ES6 Demo Modal

This modal is powered by ng1bs5 loaded via ES6 imports!

The component is imported directly from src/components/.

Import: 
import ModalModule from '@ng1bs5/modal';

angular.module('myApp', [ModalModule.name]);

Popover Component

Contextual overlays using ng1bs5 directive bs-popover.

Import: 
import PopoverModule from '@ng1bs5/popover';

angular.module('myApp', [PopoverModule.name]);

Usage: 
<button bs-popover 
        data-bs-title="Title"
        data-bs-content="Content"
        data-bs-placement="top">
    Click me
</button>

Tooltip Component

Helpful hints using ng1bs5 directive bs-tooltip.

üöÄ Getting Started with ES6

1. Install dependencies: 
npm install
2. Run development server: 
npm run dev
3. Build for production: 
npm run build
4. Import components in your app.js: 
import angular from 'angular';
import ModalModule from '@ng1bs5/modal';
import PopoverModule from '@ng1bs5/popover';

const app = angular.module('myApp', [
  ModalModule.name,
  PopoverModule.name
]);
