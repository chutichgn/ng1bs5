Usage
Basic Usage: 
<button bs-tooltip 
        data-bs-title="Tooltip text"
        data-bs-placement="top">
    Hover me
</button>
With HTML Content: 
<button bs-tooltip 
        data-bs-title="<strong>Bold</strong> text"
        data-bs-html="true"
        data-bs-placement="right">
    Hover me
</button>
Available Attributes:
  • data-bs-title - Tooltip content (required)
  • data-bs-placement - Position: 'top', 'right', 'bottom', 'left' (default: 'top')
  • data-bs-html - Enable HTML content (default: false)
  • data-bs-trigger - Trigger method: 'hover', 'click', 'focus' (default: 'hover')
  • data-bs-delay - Delay in ms before showing/hiding
  • data-bs-container - Container selector for tooltip element
