Usage
Basic Accordion: 
<bs5-accordion close-others="true">
    <bs5-accordion-group heading="Item 1" is-open="true">
        Content for item 1
    </bs5-accordion-group>
    
    <bs5-accordion-group heading="Item 2">
        Content for item 2
    </bs5-accordion-group>
</bs5-accordion>
With Custom Heading: 
<bs5-accordion-group>
    <bs5-accordion-heading>
        <i class="bi bi-star"></i> Custom Heading
    </bs5-accordion-heading>
    Content here
</bs5-accordion-group>
Attributes:
  • close-others - (boolean) Close other panels when one opens (default: true)
  • heading - Text for the accordion heading
  • is-open - (boolean) Initial open state
  • is-disabled - (boolean) Disable the accordion item
