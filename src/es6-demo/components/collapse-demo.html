Usage
ng1bs5 Directive: 
<button ng-click="isCollapsed = !isCollapsed">
    Toggle
</button>

<div bs5-collapse="isCollapsed">
    Collapsible content here
</div>
Bootstrap Native: 
<button data-bs-toggle="collapse" 
        data-bs-target="#myCollapse">
    Toggle
</button>

<div class="collapse" id="myCollapse">
    Collapsible content here
</div>
Attributes:
  • bs5-collapse - Boolean expression to control collapsed state
  • The directive automatically handles the Bootstrap classes and animations
  • Works seamlessly with AngularJS data binding
