Getting Started
1. Install dependencies: 
npm install @uirouter/angularjs
2. Import in your app: 
import uiRouter from '@uirouter/angularjs';
import ng1bs5 from 'ng1bs5';

angular.module('app', [uiRouter, ng1bs5]);
3. Configure states: 
$stateProvider.state('myState', {
  url: '/my-state',
  template: '
Content
'
});

Architecture

Tabs Component

Uses the <bs5-tabset> component we developed. Each tab wraps a <div ui-view></div> for state content.

UI-Router States

Each component demo is a separate state with its own URL. States are synchronized with tabs for seamless navigation.

Synchronization

Tab clicks trigger state changes. State changes sync tab selection. Browser back/forward buttons work perfectly.
