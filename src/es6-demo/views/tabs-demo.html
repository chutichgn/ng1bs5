API Reference
bs5-tabset Attributes:
  • tabset-id - Unique identifier for programmatic control
  • type - 'tabs', 'pills', or 'underline'
  • vertical - Display tabs vertically
  • justified - Equal width tabs
  • fill - Proportional width tabs
  • fade - Enable fade animation
  • on-select - Callback when tab is selected
bs5-tab Attributes:
  • tab-id - Unique identifier
  • heading - Tab heading text
  • active - Initial active state
  • disabled - Disabled state
