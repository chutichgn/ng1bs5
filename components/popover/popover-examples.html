Popover Component Examples

Comprehensive examples of the bs5-popover directive with various configurations.

1. Basic Popover

Simple popover with text content.

<button class="btn btn-primary" 
        bs5-popover="This is a basic popover with simple text content."
        title="Popover Title">
    Click for Basic Popover
</button>

2. Placement Options

Popovers can be positioned in different directions.

<button bs5-popover="Content" title="Title" placement="top">Top</button>
<button bs5-popover="Content" title="Title" placement="right">Right</button>
<button bs5-popover="Content" title="Title" placement="bottom">Bottom</button>
<button bs5-popover="Content" title="Title" placement="left">Left</button>

3. Trigger Options

Different ways to trigger popovers.

<button bs5-popover="Content" trigger="click">Click</button>
<button bs5-popover="Content" trigger="hover">Hover</button>
<input bs5-popover="Content" trigger="focus" />

4. HTML Content

Popovers can contain HTML content.

<button bs5-popover="<strong>Bold</strong> and <em>italic</em>"
        title="HTML Content"
        html="true">
    HTML Content Popover
</button>

5. Template URL

Load popover content from an external template.

<button bs5-popover template-url="popover-template.html">
    Template Popover
</button>

<script type="text/ng-template" id="popover-template.html">
    <div>
        <p>Template content</p>
        <button ng-click="close('data')">Close</button>
    </div>
</script>

6. Popover with Controller

Attach a controller to the popover for complex interactions.

Last Action: {{vm.lastAction}} 
<button bs5-popover
        template-url="template.html"
        popover-controller="MyController as ctrl"
        handler="vm.handleClose($data)">
    Controller Popover
</button>

<script type="text/ng-template" id="template.html">
    <div>
        <p>{{ctrl.data}}</p>
        <button ng-click="close(data)">Close</button>
        <button ng-click="dismiss(reason)">Dismiss</button>
    </div>
</script>

7. Animation Control

Control fade animation.

<button bs5-popover="Content" animate="true">With Animation</button>
<button bs5-popover="Content" animate="false">No Animation</button>

8. Delay Options

Add delays to show and hide.

<button bs5-popover="Content" delay="500">Delay 500ms</button>
<button bs5-popover="Content" delay="{show: 300, hide: 800}">Custom</button>

9. Advanced Positioning

Control offset and fallback placements.

<button bs5-popover="Content"
        placement="right"
        offset="[20, 10]"
        fallback-placements="['left', 'top', 'bottom']">
    Advanced Positioning
</button>

10. Custom Container

Render popover in a specific container.

This is a custom container with position: relative

<div id="custom-container" style="position: relative;">
    <button bs5-popover="Content" container="#custom-container">
        Popover in Container
    </button>
</div>
