This demo application uses ES6 imports to load ng1bs5 components directly from the source. It's built with Webpack and demonstrates how to use the library in modern JavaScript applications.

Modal Component

Dialog boxes and popup windows with Bootstrap 5 styling.

ES6 Demo Modal

This modal is loaded from the ng1bs5 source using ES6 imports!

The component is imported directly, not from a bundled distribution file.

Import: 
import { ModalModule } from '@ng1bs5/modal';

ðŸš€ Getting Started with ES6

1. Install dependencies: 
npm install
2. Run development server: 
npm run dev
3. Build for production: 
npm run build
4. Import components in your app.js: 
import angular from 'angular';
import { ModalModule } from '@ng1bs5/modal';
import { PopoverModule } from '@ng1bs5/popover';

const app = angular.module('myApp', [
  ModalModule.name,
  PopoverModule.name
]);
