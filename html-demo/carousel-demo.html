Component API

Attribute Type Default Description
interval number 5000 Time to delay between cycling an item (ms)
keyboard boolean true Whether carousel should react to keyboard events
pause string|boolean 'hover' Pauses cycling on hover
ride string|boolean false Autoplays the carousel after load
