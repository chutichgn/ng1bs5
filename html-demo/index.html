üöÄ Getting Started

To use ng1bs5 in your project:

  1. Include AngularJS and Bootstrap 5 CSS
  2. Include the ng1bs5 library: <script src="dist/ng1bs5.js"></script>
  3. Add the required module to your app: angular.module('myApp', ['ng1bs5.{component}'])
  4. Use the components in your templates!

Note: For ES6 module imports, check out the ES6 Demo.
